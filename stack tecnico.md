## STACK TÉCNICO
```
Frontend:  React + TypeScript + TailwindCSS
Backend:   Node.js + Express
ORM:       Prisma
DB:        PostgreSQL
```

> ⚠️ No uses librerías de componentes UI externas (MUI, Shadcn, Chakra, etc).
> Solo TailwindCSS puro para los estilos.

> ⚠️ No uses class components de React. Solo functional components con hooks.
