## RESTRICCIONES

- **No** implementes autenticación ni login. La app es de un único usuario
- **No** uses librerías de UI externas (MUI, Shadcn, Chakra, etc.)
- **No** uses CSS inline; todo estilo debe ir en clases de Tailwind
- **No** agregues módulos que no estén listados arriba
- **No** inventes campos adicionales que no estén especificados
- **No** uses class components de React
